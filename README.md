# ChainedDialogLibrary
链式对话框库，支持自定义动画

## 动态预览图
<img src="/preview/preview.gif" alt="动态预览图" width="360" height="640">

### 所有对话框都有通用的方法

|方法|作用|返回值|  
|:-|:------:|:-:|
|setShowBottom(boolean showBottom)|内容二|BaseDialog的DialogFragment|
|setSize(int width, int height)|内容四|BaseDialog的DialogFragment|
|setMargin(int margin)||BaseDialog的DialogFragment|
|setOutCancel(boolean outCancel)||BaseDialog的DialogFragment|
|show(FragmentManager manager)||BaseDialog的DialogFragment|
|setDialogAnim(DialogAnim dialogAnim)||BaseDialog的DialogFragment|
